PROJEKTNA NALOGA 2 (SVG, PNG, CANVAS)

An SVG image (Scalable Vector Graphics) is a vector-based image format described using XML. Instead of pixels, it’s made from mathematical shapes and paths, so it scales to any size without losing quality.

A PNG (Portable Network Graphics) image is a raster-based image format that uses lossless compression. It supports transparency and is ideal for images with sharp edges, text, and graphics.

A Canvas image is created using the HTML5 canvas element, which allows for dynamic, scriptable rendering of 2D shapes and bitmap images. It is raster-based and can be manipulated via JavaScript.
